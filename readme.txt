This plugin only allows a post to be in one category by changing category check boxes into radio buttons.

== Description ==

Only One Category makes sure users only select one category for their post by make the selectors radio buttons, which only allow one category to be selected.

== Installation ==

Activate the plugin. No other steps required.

== Frequently Asked Questions ==

= How does it work? =

Plugin live-swaps check boxs for radio button before the pages are sent to users.

= Are there any options? =

Not yet.
